# AI Job Radar 浏览器扩展

这是一个无需打包的 Chrome/Edge Manifest V3 扩展，只在你点击按钮后读取**当前标签页可见的文字**，并发送给本机 AI Job Radar 的 `POST /jobs/dom-capture`。

它不包含账号、Cookie、Token 或密码；不请求全站访问权限；不登录、翻页、点击、投标、发消息、签约、付款或提交交付物。

## 安装

1. 先按仓库 README 启动后端（默认 `http://127.0.0.1:8000`）。
2. Chrome/Edge 打开扩展管理页，开启“开发者模式”。
3. 选择“加载已解压的扩展程序”，目录选择本文件夹。
4. 在已经自行打开的任务详情页点击扩展图标，再点击“采集当前页并分析”。
5. 回到 AI Job Radar 网页重新评分。

扩展只允许将数据发送到 `localhost` 或 `127.0.0.1`，防止误传到远端服务。
